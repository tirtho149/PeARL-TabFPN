# BIBM 2026 paper draft — PEaRL+TabPFN head-to-head

The IEEE BIBM 2026 submission lives in `BIBM2026_PEaRL_TabPFN.tex`. It
compares the PEaRL+MLP baseline (paper-faithful reproduction of
arXiv:2510.03455) against PEaRL+TabPFN (this work's contribution: MLP
heads replaced 1:1 with a TabPFNRegressor bank).

## Filling the `\TBD` placeholders

The LaTeX has 16 `\TBD` cells in Table 1 and the discussion paragraphs.
After running `scripts/run_reproduction.py --apple-to-apple`, populate them
from `<output_dir>/reproduction_results.json`:

```python
import json
res = json.load(open("../results/reproduction_apple_to_apple/reproduction_results.json"))

# Table 1 cells
b = res["summary"]["baseline"]      # PEaRL+MLP (ours)
t = res["summary"]["tabpfn"]        # PEaRL+TabPFN (ours)

# Gene row, MLP column
print(f"{b['gene']['PCC'][0]:.4f} \\tiny{{$\\pm${b['gene']['PCC'][1]:.4f}}}")
# ... etc
```

Each `summary[variant][target][metric]` is a `[mean, std]` pair.

## Embedding the figures

Replace each `\fbox{...}` placeholder in the LaTeX with
`\includegraphics`:

```latex
\includegraphics[width=0.95\columnwidth]{../results/reproduction_apple_to_apple/figures/fig_h2h_1_metric_bars.png}
```

Figures are generated by `scripts/generate_figures.py`. The seven files are
`fig_h2h_1_metric_bars.png` through `fig_h2h_7_pathway_corr.png`.

## Compiling

```bash
cd paper/
pdflatex BIBM2026_PEaRL_TabPFN.tex
pdflatex BIBM2026_PEaRL_TabPFN.tex   # second pass for refs
```

Requires `IEEEtran.cls` (standard on most TeX Live distributions).
